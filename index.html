CERAD 카탈로그 개인정보 보호정책

최종 업데이트: 2026년 3월 5일

1. 개요

CERAD 카탈로그 앱(이하 "앱")을 이용해주셔서 감사합니다. 본 개인정보 보호정책은 앱 사용 시 수집되는 정보와 그 사용 방식에 대해 설명합니다.

2. 수집하는 정보

본 앱은 다음의 정보를 수집할 수 있습니다:

  • 기기 식별자: 앱 사용 통계 및 분석 목적
  • 즐겨찾기 데이터: 사용자 선호도 저장 (로컬 저장)
  • 앱 사용 로그: 성능 개선 및 오류 추적
  • 위치 정보: 수집되지 않음
  • 개인 식별 정보: 수집되지 않음

3. 정보 사용 목적

수집된 정보는 다음의 목적으로만 사용됩니다:

  • 앱 기능 제공 및 개선
  • 사용자 경험 향상
  • 기술적 문제 해결
  • 앱 성능 분석 및 통계

4. 정보 보안

  • 사용자 데이터는 암호화되어 저장됩니다
  • 즐겨찾기 데이터는 사용자 기기에만 저장되며 외부로 전송되지 않습니다
  • 제3자와 공유되지 않습니다
  • 앱 삭제 시 모든 로컬 데이터가 제거됩니다

5. 제3자 서비스

본 앱은 다음의 제3자 서비스를 사용할 수 있습니다:

  • Google Analytics: 앱 사용 통계 수집 (익명화됨)
  • Crash Reporting: 앱 오류 추적

이러한 서비스 제공자들은 자신의 개인정보 보호정책을 따릅니다.

6. 사용자 권리

사용자는 다음의 권리를 가집니다:

  • 앱 설정에서 데이터 수집 거부 (기능 제한 가능)
  • 저장된 즐겨찾기 데이터 삭제
  • 앱 삭제를 통한 모든 데이터 제거

7. 정책 변경

본 개인정보 보호정책은 예고 없이 변경될 수 있습니다. 변경 사항은 본 페이지에 게시됩니다.
